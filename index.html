Product Overview

왜 Insight Plus인가?

컨택센터에는 매일 수천 건의 고객 상담이 쌓이지만, 대부분 분석되지 않은 채 사라집니다. Insight Plus는 GPT-4o 기반 AI가 상담 데이터를 자동으로 분석하여 고객 의도(Intent)를 추출하고, 이를 비즈니스 가치 프레임워크에 매핑합니다.

단순한 리포팅이 아닌, TCO(총소유비용) 절감과 Leverage(고객가치 극대화)라는 두 가지 비즈니스 목표에 직결되는 실행 가능한 인사이트를 제공합니다.

Core Features

AI가 만드는 차세대 VOC 분석

회사명만 입력하면 AI가 도메인을 자동 학습하고, 상담 데이터에서 핵심 인사이트를 추출합니다.

💡

자동 컨텍스트 생성

회사명만 입력하면 AI가 업종, 서비스 특성, 상담 유형을 자동으로 리서치하여 분석 환경을 세팅합니다.

🌱

LLM 인텐트 추출

GPT-4o가 한 통화에서 복수의 고객 의도를 정밀하게 추출합니다. 단일 분류가 아닌 다중 인텐트 분석으로 놓치는 VOC가 없습니다.

🛠

카테고리 자동 설계

L1/L2/L3 3단계 분류 체계를 AI가 5분 내에 자동 생성합니다. 드래그앤드롭으로 직관적 수정이 가능합니다.

📊

6대 분석 대시보드

인텐트 분포, 트렌드, 가치 인사이트, TCO 시뮬레이션, CPX/IPC 지표, 문제 인텐트 등 핵심 뷰를 제공합니다.

🔍

4단계 Drill-down

대시보드의 모든 수치를 L2 요약 → L3 상세 → 인텐트 목록 → 원본 데이터까지 4단계로 검증할 수 있습니다.

💰

TCO 시뮬레이션

자동화, 프로세스 개선 등 시나리오별 비용 절감 효과를 실시간으로 시뮬레이션하여 의사결정을 지원합니다.

Business Value Framework

❌
Remove

불필요 비용 제거

불필요한 반복 문의, 시스템 오류로 인한 인입 등 근본 원인을 제거하여 콜 자체를 줄입니다.

✍️
Simplify

프로세스 간소화

복잡한 절차, 불명확한 안내를 개선하여 상담 시간을 단축하고 처리 비용을 절감합니다.

⚙️
Automate

자동화 전환

챗봇, IVR 등으로 전환 가능한 단순 반복 문의를 식별하여 운영 비용을 구조적으로 절감합니다.

🚀
Leverage

매출 기회 극대화

업셀, 크로스셀, 이탈 방지 등 숨겨진 비즈니스 기회를 발굴하여 컨택센터를 수익 창출 채널로 전환합니다.

🎯

Remove + Simplify + Automate = TCO 절감   |   Leverage = 매출 극대화
